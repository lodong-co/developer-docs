# 케이스 스터디 — 무인 스토리지 (자체 제작 스마트 사물함 · LoRa)

> **사물함을 직접 제작**하고, LoRa 무선으로 게이트웨이와 통신해 **앱으로 원격 개폐**하는 무인 보관 시스템

## 도전

무인 보관(스토리지) 서비스를 운영하려면 다수의 사물함을 안정적으로 원격 제어해야 합니다. 그런데 사물함 하나하나에 인터넷 배선을 까는 것은 비현실적이고, 시중 완제품에 의존하면 통신·제어 방식을 통제할 수 없습니다. **하드웨어부터 통신 방식, 게이트웨이, 백엔드, 앱까지 직접 설계**해야 했습니다.

## 해결

사물함 하드웨어 · 무선 통신 · 게이트웨이 · 백엔드 · 앱을 **전 계층 자체 구축**했습니다.

### 자체 제작 스마트 사물함 + LoRa
- 잠금장치 · 제어 보드를 **직접 설계·제작** — 완제품 의존 없이 하드웨어 통제
- 각 사물함이 **저전력 장거리 무선(LoRa)** 으로 게이트웨이와 통신 — 개별 인터넷 배선 없이 넓은 범위 커버

### 게이트웨이 (전용 서비스)
- **API와 LoRa 디바이스를 잇는 전용 게이트웨이**(Go) — 개폐 명령 하달 · 사물함 상태 수집

### 백엔드 (Spring Boot)
- **크기별 사물함** 조회 · 신청 · **이용 연장**
- **카카오 기반 모바일 인증**
- 결제(이용권) · 약관 · 공지

### 앱 원격 제어
- 사용자가 **앱으로 사물함 문을 원격 개폐** — 이용 · 결제와 연동

## 결과
- **하드웨어 → LoRa → 게이트웨이 → 서버 → 앱**까지 풀스택 자체 구축
- LoRa로 **배선 없이 다수 사물함 원격 제어**
- 크기별 사물함 신청 · 이용 연장 · 결제까지 갖춘 **운영형 무인 보관 서비스**

## 기술
자체 제작 하드웨어(잠금·제어 보드) · **LoRa** · **Go 게이트웨이** · Spring Boot API · 카카오 로그인 · 모바일 앱
