# 케이스 스터디 — 무인 포토 키오스크 (강촌레일바이크)

> 관광지 현장의 **무인 즉석 사진 인화 키오스크** — 결제 + 전문 포토프린터 하드웨어 제어

## 도전

관광지(강촌레일바이크) 방문객이 직원 없이 **직접 촬영 → 결제 → 즉석 인화**까지 마칠 수 있어야 합니다. 일반 프린터가 아닌 **전문 포토프린터(DNP DS620)** 를 안정적으로 제어해야 하고, 용지·리본이 떨어지면 미리 감지해 현장 운영이 멈추지 않아야 합니다.

## 해결

전용 데스크톱 키오스크 + 모듈화된 인쇄 엔진으로 구축했습니다.

- **WPF 데스크톱 키오스크** (.NET) — 현장 무인 단말 UI
- **인터페이스 기반 PrintModule** — 인쇄 엔진 · 이미지 렌더링 · 배치 인쇄를 인터페이스로 추상화한 재사용 모듈
- **프린터 소모품 감지** — 용지/리본 잔량을 모니터링해 소진 전 대응
- **전문 포토프린터(DNP DS620) 제어** — 사진 품질 인화 하드웨어 직접 연동
- **NHN KCP 결제** — 현장 결제 통합

## 결과
- 직원 없는 **무인 즉석 사진 인화**
- 전문 포토프린터 **하드웨어 제어 + 결제**를 하나의 단말로 통합
- 인쇄 로직을 **재사용 모듈**로 분리 — 다른 키오스크에 재활용 가능

## 기술
C# · .NET WPF · DNP DS620 포토프린터 · NHN KCP
