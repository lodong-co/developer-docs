# ld-jwt-member-module — 회원 인증

JWT 기반의 회원 가입·로그인·인증 모듈. **일반 로그인·소셜 로그인·계정 연동**을 포괄하며, 호스트 앱이 회원 엔티티를 소유하는 구조로 설계되었습니다.

> 실운영: 다수 서비스에서 라이브 가동 중.

## 주요 기능
- **JWT 인증** — 액세스/리프레시 토큰 발급·검증
- **리프레시 토큰 회전 + 재사용 탐지** — 탈취된 토큰 재사용을 블랙리스트로 차단
- **일반 / 소셜 / 연동** — 아이디·비밀번호 로그인, 소셜 로그인, 계정 연동을 하위 모듈로 제공
- **안전한 쿠키 처리** — 리프레시 토큰을 HttpOnly·Secure 쿠키로
- **기동 시 비밀 검증** — JWT 시크릿 길이 등 보안 요건을 미달 시 기동 중단

## 설계 노트
- **엔티티 비소유 패턴** — 회원 엔티티의 소유권은 **호스트 앱**이 갖습니다. 모듈은 인터페이스와 콜백을 통해 가입·로그인 로직을 수행하되, 호스트가 자신의 `Member` 모델을 그대로 유지합니다. 공유 인증 라이브러리에서 가장 어려운 "데이터 모델 결합"을 제거한 설계입니다.
- **호스트 충돌 방지** — 자동 설정 순서를 제어해 호스트 앱의 JPA 저장소와 충돌하지 않습니다.

## 도입
```gradle
implementation "com.lodong:ld-jwt-member-module"   // 저장소 접근·버전은 계약 시 안내
```
