# ld-utils-module — 공통 유틸리티

전 서비스가 공유하는 **공통 토대 모듈**. 표준 응답·예외 처리·암호화·라이선스 게이트 등 모든 서비스에 반복되는 기반을 제공합니다.

> 실운영: **전 서비스 공통** (모든 lodong 모듈의 기반).

## 주요 기능
- **표준 응답 봉투** — 모든 API 응답을 일관된 형식으로 (성공/오류, 페이지네이션 포함)
- **전역 예외 처리** — 공통 예외를 한 곳에서 처리하고, 로그는 자사 프레임만 추려 노이즈를 줄임. 호스트 앱이 재정의 가능
- **암호화(AES-GCM)** — 인증 암호화(AEAD), 암호화마다 임의 IV 사용
- **라이선스 기동 게이트** — Ed25519 서명 검증으로 라이선스가 유효할 때만 모듈이 기동 (구독 상용화 기반)

## 설계 노트
- 다른 모든 모듈이 의존하는 **단일 공통 기반**으로, 응답·예외·보안 표준을 서비스 전반에 통일합니다.

## 도입
```gradle
implementation "com.lodong:ld-utils-module"   // 저장소 접근·버전은 계약 시 안내
```

> 본 문서는 기능 소개입니다. API 상세·예제·소스는 구독 계약 시 제공됩니다.
