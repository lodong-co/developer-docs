# ld-chat-module — 실시간 채팅

STOMP(WebSocket) 기반의 실시간 채팅 모듈. 단순 메시지 전송을 넘어, **종단간 암호화와 채널 레벨 권한**까지 갖춘 프로덕션급 채팅 인프라입니다.

> 실운영: **사주J**, **Provio** 에서 라이브 가동 중.

## 주요 기능
- **STOMP 실시간 송수신** — WebSocket 기반 양방향 메시징
- **종단간 암호화(E2EE)** — 봉투키(envelope-key) 프로토콜. 수신자 집합이 채팅방 참여자와 정확히 일치하는지 검증하고, 메시지 수정 시 암호화 IV(nonce) 재사용을 거부합니다.
- **채널 레벨 권한** — 사용자가 참여하지 않은 방의 구독/발신을 STOMP 채널 단계에서 차단. 컨트롤러 검증에만 의존하지 않습니다.
- **영속 우선(persistence-first)** — 브로드캐스트 전에 메시지를 먼저 저장해 유실을 방지
- **커서 페이지네이션** — 대용량 메시지 백로그를 효율적으로 조회
- **읽음 처리 / 안읽음 수** — 마지막 읽은 메시지 기반
- **첨부 파일 라이프사이클** 관리

## 설계 노트
- **엔티티 비소유** — 채팅 사용자 정보는 호스트 앱이 SPI(`ChatUserResolver`)로 제공합니다. 모듈이 호스트의 회원 모델을 침범하지 않습니다.
- **보안 우선** — E2EE 봉투키 검증, IV 재사용 거부 등 암호학적 안전성을 설계 단계에서 반영했습니다.

## 도입
```gradle
implementation "com.lodong:ld-chat-module"   // 저장소 접근·버전은 계약 시 안내
```
멀티 데이터소스(사이드카 DB) 구성 및 nginx WebSocket 프록시 연동 가이드를 함께 제공합니다.

> 본 문서는 기능 소개입니다. API 상세·예제·소스는 구독 계약 시 제공됩니다.
