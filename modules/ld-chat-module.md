# ld-chat-module — 실시간 채팅

STOMP(WebSocket) 기반의 **프로덕션급 실시간 채팅 모듈**. 채팅방 · 메시지 · 참여자 · 읽음 · 타이핑 · 첨부 · 재연결 동기화에 더해, 본문 **종단간 암호화(E2EE)** 와 채널 레벨 권한까지 갖춘 완결형 채팅 인프라입니다.

> 실운영: **사주J**, **Provio** 라이브 가동 중.

## 핵심 가치
- **신뢰성** — 장애 상황에도 메시지 유실 없음
- **순서 일관성** — 모든 참여자가 동일한 메시지 순서를 봄
- **보안** — 미참여자 도청 차단 + 본문 종단간 암호화
- **느린 업로드 내성** — 첨부 업로드가 느려도 대화 순서가 막히지 않음

## 기능

### 채팅방
- 1:1(DIRECT) · 그룹(GROUP) 채팅방
- 같은 두 사용자는 항상 하나의 1:1 방을 공유
- 권한 기반 삭제 + 메시지·참여자 이력 보존

### 메시징
- 실시간 송수신 · **다중 디바이스 동시 지원**
- 모든 참여자에게 **동일하고 결정적인 메시지 순서**
- 양방향 페이징(과거 조회 / 증분 동기화) · 본인 메시지 삭제(자리표시자)
- 옵티미스틱 UI 지원 (즉시 표시 후 서버 확정 매칭)

### 참여자 · 상태
- 초대 / 퇴장 / 알림 음소거 (권한 제어)
- 읽음 영수증 + 읽음 표시 실시간 반영
- 타이핑 인디케이터
- 방별 · 전체 미읽음 카운트 (글로벌 뱃지)

### 오프라인 · 재연결
- 재연결 시 **오프라인 동안 놓친 메시지를 자동 동기화** — 누락 없는 복귀

### 첨부
- 이미지 · 동영상 · 파일 첨부 (메시지당 다중)
- **업로드 상태 표시** — 업로드 중 / 완료 / 실패
- 느린 업로드가 대화 순서를 막지 않음

### 종단간 암호화 (E2EE)
- 메시지 본문을 **종단간 암호화로 보호** — 서버 · DB 관리자 · 모듈도 평문에 접근 불가
- **점진 도입**(평문 공존) ~ **전면 강제** 모드 지원
- 키 발급 · 회전 지원

## 설계 노트
- **엔티티 비소유** — 사용자 정보는 호스트 앱이 제공, 호스트의 회원 모델을 침범하지 않음
- 멀티 데이터소스 · WebSocket 프록시 등 운영 연동 가이드 제공
