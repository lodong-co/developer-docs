# ld-message-module — 쪽지 · 메시지

사용자 간 **쪽지 · 메시지** 기능 모듈. 메시지 본문을 암호화하여 저장합니다.

## 주요 기능
- **쪽지 · 메시지 송수신** — 사용자 간 메시지
- **암호화 저장** — 메시지 본문을 암호화하여 보관
- **읽음/상태 관리** — 메시지 상태 처리

## 설계 노트
- 암호화 검증을 포함한 **충실한 테스트 커버리지**(단위 + 통합 테스트)를 갖춘 모듈입니다.

## 도입
```gradle
implementation "com.lodong:ld-message-module"   // 저장소 접근·버전은 계약 시 안내
```

> 본 문서는 기능 소개입니다. API 상세·예제·소스는 구독 계약 시 제공됩니다.
