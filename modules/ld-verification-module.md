# ld-verification-module — 검증 플로우

검증 절차를 **상태머신(FSM)** 으로 관리하는 모듈. 검증 상태의 전이 규칙과 종료 상태를 엄격하게 통제합니다.

## 주요 기능
- **상태머신 기반 검증** — 검증 단계를 명시적 상태 전이로 관리
- **전이 규칙 강제** — 허용된 전이만 가능, 잘못된 전이 차단
- **종료 상태 잠금** — 완료/실패 등 종료 상태에서는 추가 변경 불가

## 설계 노트
- 검증 상태 전이 매트릭스와 종료 상태 잠금을 **테스트로 검증**합니다. 상태 일관성이 중요한 도메인(본인확인·승인 절차 등)에 적합합니다.

## 도입
```gradle
implementation "com.lodong:ld-verification-module"   // 저장소 접근·버전은 계약 시 안내
```
