# ld-membership-module — 멤버십

회원의 **멤버십 · 등급**을 관리하는 모듈. 등급 체계와 멤버십 상태를 다룹니다.

## 주요 기능
- **멤버십 · 등급 관리** — 회원 등급 체계와 상태
- **회원 도메인 연동** — 인증/회원 모듈과 함께 동작 (엔티티 소유권은 호스트 앱)

## 설계 노트
- lodong 모듈 중 **가장 규모가 큰 도메인 모듈**로, 멤버십 관련 기능을 폭넓게 다룹니다.
- 회원 엔티티 소유권은 호스트 앱이 갖는 **엔티티 비소유 패턴**을 따릅니다.

## 도입
```gradle
implementation "com.lodong:ld-membership-module"   // 저장소 접근·버전은 계약 시 안내
```

> 본 문서는 기능 소개입니다. API 상세·예제·소스는 구독 계약 시 제공됩니다.
