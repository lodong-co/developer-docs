# ld-seo-module — SEO · 색인 자동화

콘텐츠를 발행하면 **검색엔진·생성형 AI 노출에 필요한 것들이 자동으로** 만들어지는 SEO 모듈. "글만 쓰면 SEO 자동"을 백엔드에서 책임집니다.

> 실운영: **콕컷(kokcut.com)** 라이브 가동 중.

## 핵심 가치
- **자동화** — 발행 한 번이 메타 · 사이트맵 · 색인 통지까지 이어짐
- **다중 채널** — 구글 · 네이버 · 빙 + 생성형 AI 검색(GEO)을 함께 겨냥
- **무손 관리** — 새 글이 사이트맵·색인에 자동 반영(수동 작업 0)

## 기능
### 메타 자동 생성
- 제목·본문에서 **slug · 메타 타이틀 · 설명(요약) · 키워드 · canonical · JSON-LD 타입** 자동 산출
- 한글 slug 지원(UTF-8)

### 색인 통지 (IndexNow)
- 발행 즉시 **빙 · 네이버 · 얀덱스에 색인 요청**(자동, 비동기)
- 구글은 사이트맵 + 크롤링 경로 지원

### 사이트맵
- 발행 콘텐츠를 사이트맵 엔트리로 노출(여러 제공자 합산)

### 확장점(SPI)
- **요약기** — 기본 룰 기반, LLM 요약 구현으로 교체 가능
- **OG 이미지 생성기** — 기본 미사용, AI 서버 이미지 생성으로 교체 가능

## 설계 노트
- **엔티티 비소유** — 콘텐츠 엔티티는 호스트 앱 소유, 모듈은 메타 생성·색인 통지 로직만 제공
- 프론트 짝꿍 `@lodongwebmodule/seo`(Next.js metadata · JSON-LD · sitemap · robots · llms · 소유권 인증)과 결합
